//
// Texture & Normal & Displacement mapping shader
//
///////////////////////////////////////////////////////////////////////////////////////////////////////

- This shader applies texture + normal + displacement mapping to a geometry.

- It uses Hull and Domain shader to apply tessellation. The primitive topology used is patch of 3 control points.

- It renders geometry to 4 different Render Target Views (normal, diffuse albedo, specular albedo, position), for deferred rendering purposes.
